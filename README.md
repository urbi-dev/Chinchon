# Assets

Icon assets are generated from `assets/brand/icon.svg` via `scripts/generate-icons.mjs`.

Commands:

```bash
npm run gen:icons
```

Outputs:
- Web: `assets/web-icons/icon-{16..512}.png`
- Android (Play Store): `assets/android/store-icon-512.png`, `assets/android/feature-graphic-1024x500.png`
